# Code of Conduct

## Our Commitment

We are committed to fostering a welcoming, inclusive, and respectful environment for everyone who participates in the Scotec.Revit project, whether as a contributor, maintainer, or user.

## Expected Behavior

- Be respectful and considerate in all interactions.
- Provide constructive feedback and criticism.
- Communicate openly and thoughtfully.
- Respect differing viewpoints and experiences.
- Help create a collaborative and supportive environment.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination of any kind.
- Offensive, derogatory, or abusive language or behavior.
- Personal attacks or insults.
- Unwelcome sexual attention or advances.
- Disruptive behavior that interferes with project collaboration.

## Reporting Issues

If you experience or witness unacceptable behavior, please report it to the project maintainers via [GitHub Issues](https://github.com/scotec-Software-Solutions-AB/scotec-revit/issues) or by contacting a maintainer directly.

All reports will be handled confidentially and promptly.

## Enforcement

Project maintainers have the right and responsibility to remove, edit, or reject comments, commits, code, issues, and other contributions that do not comply with this Code of Conduct, and may ban any participant for violations.

## Scope

This Code of Conduct applies to all project spaces, including GitHub repositories, issue trackers, pull requests, and any other communication channels related to the Scotec.Revit project.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/) version 2.1.
